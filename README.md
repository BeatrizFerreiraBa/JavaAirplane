# ✈️ JavaAirplane

Sistema desenvolvido em Java com foco na simulação do gerenciamento de uma companhia aérea, permitindo o cadastro e gerenciamento de informações relacionadas aos voos.

---

## 📖 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de Programação Orientada a Objetos (POO) utilizando Java.

Durante o desenvolvimento foram aplicados conceitos como:

- Encapsulamento
- Herança
- Polimorfismo
- Classes e Objetos
- Coleções
- Tratamento de Exceções
- Organização em pacotes

---

## ✨ Funcionalidades

- Cadastro de aviões
- Cadastro de voos
- Cadastro de passageiros
- Consulta de informações
- Gerenciamento das operações

## 🚀 Como executar

Clone o repositório

```bash
git clone https://github.com/BeatrizFerreiraBa/JavaAirplane.git
```

Entre na pasta

```bash
cd JavaAirplane
```

Compile

```bash
javac Main.java
```

Execute

```bash
java Main
```
